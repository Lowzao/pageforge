
📄 FORMATO DO ARQUIVO .TXT DE PAÍSES (para gerar 1 PDF com várias páginas)

Cada linha representa os dados para 1 página do PDF.

Formato:
campo1;campo2;...

Exemplo:
Brasil;América do Sul
França;Europa
Japão;Ásia

E na área "Coordenadas", você define onde cada campo aparece na página:
nome=100,500
continente=100,450
